# 项目简介 java970

本项目是基于 Spring Boot 和 Vue 的协同过滤管理系统，采用前后端分离的技术架构。

## 技术栈

- Java
- Spring Boot
- MySQL
- MyBatis
- Maven
- Vue

## 功能模块

- 系统管理员管理
- 员工管理
- 财务管理
- 协同过滤算法实现

## 系统角色

- 系统管理员
- 员工
- 财务人员

## 运行环境

- JDK 1.8
- Maven
- MySQL 5.7
- Node.js 14 版本

## 部署步骤

1. 准备运行环境：安装 JDK 1.8、Maven、MySQL 5.7 和 Node.js 14 版本。
2. 导入数据库。
3. 使用 Maven 构建后端项目。
4. 使用 Node.js 运行前端项目。

## 目录结构

```
/backend  -- 后端代码
/frontend -- 前端代码
```

## 核心亮点

1. 前后端分离，易于维护和扩展。
2. 采用成熟的协同过滤算法，提高管理效率。
3. 经过严格测试，确保源代码质量。
4. 适用于学习和实际应用场景。
